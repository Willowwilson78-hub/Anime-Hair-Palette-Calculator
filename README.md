# Anime-Hair-Color-Calculator
[![image](https://github.com/user-attachments/assets/6a7dcb81-edab-47df-929e-734068e34831)]([https://example.com](https://willowwilson78-hub.github.io/Anime-Hair-Color-Calculator/)

A lightweight, simple, real-time web tool to calculate dynamic highlights and shadows for anime hair palettes based on a base color selection. Built with pure HTML5 Canvas and CSS Glassmorphism.
